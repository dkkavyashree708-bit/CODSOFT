# Task 1: To-Do List Application

## 📌 Project Overview
This is a command-line based To-Do List application developed using Python in PyCharm.
It allows users to manage daily tasks efficiently through a simple interface.

---

## 🚀 Features
- Add new tasks
- View existing tasks
- Mark tasks as completed
- Delete tasks
- File handling for saving tasks permanently



## 🛠 Technologies Used
- Python
- PyCharm IDE
- File Handling
- Basic OOP Concepts



 How to Run the Program

### Using PyCharm:
1. Open the project in PyCharm.
2. Open `todo.py`.
3. Click the Run button ▶.

### Using Command Prompt (Optional):
```bash
python todo.py
```
